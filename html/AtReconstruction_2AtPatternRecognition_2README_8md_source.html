1 # trackfinder - Program for finding particle tracks in point clouds
2 
3 ## About
4 
5 trackfinder is a program that clusters points from a point cloud into
6 tracks, which can overlap at vertex points. It assigns each point a track
7 label (0, 1, ...) or the label -1 for noise. Points belonging to more than
8 one track obtain a combined label with teh individual labels separated
9 by semikolons.

27 ## Usage
28 
29 When called without any arguments, `trackfinder` gives a usage message.
30 
31 The input file must be in PCL data format (this will possibly be changed in
32 future versions). To convert from AtTPC data file to PCL file format, there
33 is an external python script (not included in these source package):
34 
35  $ python3 dat2pcl.py infile outfile
36 
37 The output is one of the following:
38 
39  - CSV file (default) with two lines header (starts with #)
40  followed by one point per line with its track label
41  ("-1" for noise, "L1;L2" for points belonging to L1 and L2)
42  field separator is a comma
43  - gnuplot command for drawing the result with option `-gnuplot`
44  - screen display of result with option `-x11`
45 
46 Example:
47 
48  $ trackfinder /path/to/infile -r 5 -gnuplot | gnuplot -persist
